# Void Enemies Revamped

### General

- Reavers, Jailers, and Devastators are now immune to void death
- Reavers, Jailers, and Devastators can no longer be hitstunned (a high damage hit can't stun them)

### Void Reaver

- Portal bomb is now predictive
- Portal bomb can now spawn mid air
- Portal bomb spawns in clusters of 3
- Portal bomb doesn't apply nullify stacks if already nullified (prevents perma-root)
